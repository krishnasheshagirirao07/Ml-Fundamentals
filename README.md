Python ML Fundamentals
A collection of Python notebooks exploring core concepts in data manipulation, numerical computing, and data visualisation — the building blocks of machine learning workflows.

What's Inside
Python Basics

Variable operations and mathematical expressions (e.g. computing 2x + e³)
Defining and calling custom functions

Data Structures

Working with lists, loops, and reverse iteration
Building and querying dictionaries
Using f-strings for formatted output

NumPy

Creating and manipulating multi-dimensional arrays
Matrix multiplication with np.matmul()
Array slicing and indexing

Data Visualisation

Loading .npy datasets with NumPy
Scatter plots using Matplotlib
Axis labelling and chart titles


Dataset
Uses the classic Fisher Iris Dataset — 150 samples of iris flowers with 4 measured features each (sepal length, sepal width, petal length, petal width). A scatter plot of sepal length vs petal length is generated to explore the data visually.
